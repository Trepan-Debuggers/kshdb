Info Program
------------

**info program**

Execution status of the program. Listed are:

* Reason the program is stopped.
* The next line to be run

Example:
++++++++

::

     kshdb<1> info program
     Program stopped.
     It stopped after being stepped.
     Next statement to be run is:
     [ "${PS1-}" ]
